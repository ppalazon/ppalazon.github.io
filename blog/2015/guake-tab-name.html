Tab name on Guake

Don't let guake rename tab name automatically

Posted by Pablo Palaz贸n on January 06, 2015

If you use guake terminal with so many tabs, you will found a problem if guake rename you tab name automatically. Guake terminal rename tab name with user@host and directory path

You can resolve it, with a gnome configuration property. Open the terminal and write

gconftool-2 --set /apps/guake/general/use_vte_titles --type boolean false

Then, guake let fixed the terminal name
